SplitSearch is a better version of 'Search Web for XXX'.

It eliminates research friction by instantly locking your current page and a persistent, dedicated search window into a 50/50 split-screen environment. 

How to Deploy: Download, Put it in a folder, and upload unpacked in your browser's extension
How to Use: Select the text, Right-click, and choose 'Search in Focus Window' 
Usecase: when you're reading document/webpage but want to search something but keep your attention.
Browser: Chrome and Edge are both okay for this version.
